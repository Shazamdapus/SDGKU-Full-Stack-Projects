This Is A Sample Page Showing Off CSS!!

This is an example of how you use internal CSS. Although it is effective,
it makes changing style in a web page cumbersome and muddies up the html code.
Best practice is to use and external style sheet.

External Style Sheets Are Way Better!!

When it comes to setting style for your web page, external style
sheets are the best option. It helps keep the functional html code clear
but also allows the programmer easy access to change the style of the page.
Using an external stylesheet also allows you to apply the same style
to multiple pages by simply just linking the sheet to the other pages.

It also allows you to change the style of individual elements in the page
just as long as you set up the id's correctly. See how much different
this paragraph is to the one above it? Thats because I used a different
id and linked that to a different style on my CSS. The power of the CSS
is strong with this one!!
